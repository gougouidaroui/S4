Généralités

1.1 Définition

L'empreinte est un enregistrement en négatif de la topographie d'une région de la cavité buccale ou d'un modèle (techniques indirectes).

Moule (Négatif)
→

→
Modèle (Positif)

Son moulage permet d'obtenir une réplique en positif.

1.2 Objectif

L'objectif de l'empreinte est la reproduction de toute ou une partie des arcades dentaires afin d'obtenir des modèles de travail.

2.1 Les Matériaux Rigides

2.1.1 Le Plâtre

Indications

  • Matériaux de confection des modèles d'étude et de travail
  • Le plâtre à empreinte est utilisé pour enregistrer de manière statique les surfaces muqueuses de l'édenté total

Composition

Le gypse constitue la matière première du plâtre. Ce dihydrate de sulfate de calcium (CaSO₄), déshydraté par une cuisson au four, se transforme en plâtre.

Gypse
CaSO₄·2H₂O
→
Cuisson
→
Plâtre
CaSO₄·½H₂O
Type Désignation Caractéristiques
Type I Plâtre pour empreinte Usage spécifique
Type II Plâtre dentaire (Paris) Fragile
Type III Plâtre pierre Dur
Type IV Plâtre pierre amélioré Extra dur

Réaction de Prise

1. Eau dans bol
→
2. Verser plâtre
→
3. Spatulation 30s
→
4. Consistance 1-2 min
2(CaSO₄·½H₂O) + 3H₂O → 2(CaSO₄·2H₂O) + chaleur

Facteurs Influençant le Temps de Prise

Augmente le temps

  • Eau froide
  • Retardateurs (carbonate K/Na)
  • ↑ Rapport eau/poudre

Diminue le temps

  • Malaxage rapide
  • Accélérateurs (sulfate K)
  • ↓ Rapport eau/poudre

Propriétés

− Résistance: Faible + Précision: Bonne + Stabilité: Très bonne + Hydrophile

2.2.2 Les Élastomères

Les élastomères sont des caoutchoucs synthétiques, substances macromoléculaires possédant des propriétés élastiques caractéristiques.

Avantages

  • Toutes techniques d'empreintes
  • Toutes situations cliniques
  • Pas d'effets secondaires
  • Compatibles avec tous matériaux

Les 3 Familles d'Élastomères

Polyéthers
Silicones

2.2.2.2 Les Polyéthers

Produit de base : Tétra-méthylène glycol avec groupe terminal imine

Généralement monophase (ex: Impregum)

Indications

  • Reconstructions corono-radiculaires
  • Préparation cavitaire peu profonde
  • Dents sans contre-dépouilles importantes

Propriétés

+ Dureté: Très élevée + Stabilité: Satisfaisante + Précision: Grande + Prise: Rapide + Hydrophile
2.2.2.3 Les Silicones

Composé d'une longue chaîne macromoléculaire : alternance d'atomes de silicium et d'oxygène.

Deux Types de Polymérisation

🔥 Condensation

Élimination de petites molecules (eau)

Matériau: Poly-dimethylsiloxanes

Indications: Recouvrement total, limite intra-sulculaire

✨ Addition

Sans élimination de molecules

Matériau: Vinyl polysiloxanes

Indications: Facette, inlay, onlay

Silicones par Addition (VPS) - Propriétés

+ Plusieurs viscosités + Stabilité: Grande + Mouillabilité + Résistance déformation + Hydrophile + Compatible matériaux réplique

Conclusion

Il n'existe pas de matériau d'empreinte idéal et universel.
